# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately to
**security@pantalasa.org**. Do not open a public issue for security reports.

Include as much detail as possible:

- A description of the issue and its impact
- Steps to reproduce
- Affected versions or components

We aim to acknowledge reports within 2 business days and provide a remediation
timeline after triage.

## Supported Versions

The default branch is actively maintained and receives security fixes.
