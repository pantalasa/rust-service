## Project Overview

The rust-service component provides the "rust-demo" capability for the Pantalasa
platform. It is owned by carlos@pantalasa.org and implemented primarily in rust.

## Installation

Clone the repository and install dependencies using the standard rust
toolchain.

## Usage

Build and run using the commands documented in [AGENTS.md](AGENTS.md).
